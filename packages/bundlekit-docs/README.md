# bundlekit-docs

bundlekit 官方文档站，基于 [dumi](https://d.umijs.org) 构建。

覆盖快速开始、配置参考、CLI、打包器适配器（Webpack / Vite / Rollup / Rspack / Rolldown / Parcel / esbuild）、插件系统（plugin-react / plugin-vue / plugin-node / plugin-mock）、SSR 指南、AI Agent 集成（MCP）、架构设计与贡献指南。

## Development

```bash
# install dependencies
$ pnpm install

# start dev server
$ pnpm start

# build docs
$ pnpm run build

# Locally preview the production build
$ pnpm run preview
```
